# BlissGems

BlissGems is a feature-rich Minecraft plugin that introduces a sophisticated gem system with unique abilities, energy mechanics, and progression systems. Each gem offers distinct powers that players can harness and upgrade through gameplay.

## Overview

BlissGems adds 8 unique gem types to your server, each with special abilities that can be activated and upgraded. The plugin features an advanced energy system where gems can exist in various states from BROKEN to PRISTINE+5, providing players with meaningful progression and maintenance mechanics.

## Key Features

### Eight Unique Gems
Each gem type offers distinct abilities:
- **Astra** - Teleportation and mobility enhancement
- **Fire** - Offensive fire-based abilities
- **Flux** - Time and space manipulation
- **Life** - Healing and regeneration powers
- **Puff** - Cloud and air-based abilities
- **Speed** - Enhanced movement capabilities
- **Strength** - Combat power amplification
- **Wealth** - Economic and fortune bonuses

### Energy System
Gems use an energy-based durability system with 11 distinct states:
- Energy levels range from 0% (BROKEN) to 150% (PRISTINE+5)
- Gems degrade with use and can be repaired
- Higher energy states provide enhanced abilities
- Visual feedback for energy levels

### Oraxen Integration
Seamless integration with Oraxen for custom items:
- Custom gem textures and models
- Unique item properties
- NBT data support
- Resource pack compatibility

### Fully Customizable
Complete control over plugin behavior:
- Configurable ability cooldowns
- Adjustable energy consumption rates
- Customizable gem drop rates
- Flexible permission system

## Quick Start

Get started with BlissGems in just a few steps:

1. **Install the Plugin**
   - Download BlissGems.jar
   - Place in your server's `plugins/` folder
   - Install Oraxen (required dependency)
   - Restart your server

2. **Configure Oraxen Items**
   - Add gem item definitions to Oraxen
   - Configure custom models and textures
   - Restart to apply changes

3. **Test Gem Abilities**
   - Use `/blissgems give <player> <gem>` to obtain gems
   - Right-click to activate abilities
   - Monitor energy levels with `/blissgems info`

4. **Customize Settings**
   - Edit `config.yml` to adjust abilities
   - Configure energy consumption rates
   - Set permission levels
